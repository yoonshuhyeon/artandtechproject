# 썸웨워 (Yo:Aengting) - 실시간 과팅 도우미

**썸웨워**은 과팅 현장에서 어색함을 깨고, 원활한 진행과 설레는 매칭을 도와주는 실시간 웹 애플리케이션 프로토타입입니다.

--

## ✨ 주요 기능

### 1. 실시간 방 관리 및 참여
- **방 생성 및 관리**: 대표(방장)가 장소, 시간, 인원수를 설정하여 방을 생성합니다.
- **간편 입장 시스템**: 4자리 룸 코드, 초대 링크 복사, 또는 실시간 생성되는 QR 코드를 통해 팀원들이 빠르게 입장할 수 있습니다.
- **실시간 상태 동기화**: 2초 간격의 폴링(Polling)을 통해 현재 참여 인원(남/녀 수)과 매칭 진행 상태를 실시간으로 확인합니다.
- **세션 유지**: `localStorage`를 활용하여 브라우저를 새로고침하더라도 기존 입장 상태가 유지됩니다.

### 2. 고도화된 매칭 (사랑의 짝대기) 알고리즘
참여자 전원이 투표를 완료하면, 단순 랜덤이 아닌 아래의 우선순위에 따라 최적의 커플을 매칭합니다.
1. **서로 지목 (Mutual Match)**: 서로를 선택한 남녀를 최우선으로 매칭합니다.
2. **여자 우선 (Female Choice)**: 남은 인원 중 여자가 선택한 남자가 아직 매칭 전이라면 매칭합니다.
3. **남자의 선택 존중 (Concentration Handling)**: 여러 여자가 한 남자를 지목했을 경우, 그 남자가 지목한 여자와 매칭하여 선택권을 존중합니다.
4. **잔여 인원 랜덤 매칭**: 지목이 엇갈리거나 선택하지 않은 경우 무작위로 매칭을 완료합니다.

### 3. 실시간 도우미 & 아이스브레이킹
- **분위기별 질문 카드**: '가벼운/깊은/재미있는' 카테고리별 질문을 제공하여 대화가 끊기지 않게 도와줍니다.
- **SOS 긴급 요청**: 화장실 가기, 지인 호출 등 곤란한 상황에서 티 나지 않게 도움을 요청하는 기능을 시뮬레이션합니다.
- **벌칙 룰렛 & 게임**: 밸런스 게임, 질문 카드 게임 등 분위기를 띄울 수 있는 콘텐츠를 제공합니다.

## 🛠 기술 스택

### Backend
- **Framework**: Python (FastAPI)
- **Data Persistence**: 
  - 기본: 서버 메모리 (`dict`)
  - 확장: **Upstash Redis** (환경변수 설정 시 데이터 영속성 지원)
- **Deployment**: Vercel 최적화 (`vercel.json` 설정 가능)

### Frontend
- **Structure**: Single Page Application (SPA) 구조
- **Language**: HTML5, Vanilla CSS, Vanilla JavaScript
- **Real-time**: 주기적 API 폴링을 통한 상태 업데이트

## 📁 프로젝트 구조

```text
/
├── api/
│   └── main.py          # FastAPI 서버, API 엔드포인트 및 매칭 로직
├── static/
│   ├── index.html       # SPA 메인 구조 및 View 정의
│   ├── script.js        # 프론트엔드 비즈니스 로직 및 API 연동
│   └── style.css        # UI 스타일링 (Pink 테마)
├── assets/              # UI용 이미지 및 아이콘 리소스
├── requirements.txt     # 프로젝트 의존성 라이브러리
└── README.md            # 프로젝트 문서
```

---

## 🏃 실행 방법

1. **의존성 설치**
   ```bash
   pip install -r requirements.txt
   ```

2. **서버 실행**
   ```bash
   uvicorn api.main:app --reload
   ```

3. **접속**
   - 브라우저에서 `http://localhost:8000` 접속

## 📝 참고 사항
- 본 프로젝트는 프로토타입으로, 현재 데이터는 서버 메모리(`ROOMS` dict)에 저장됩니다. 서버 재시작 시 데이터가 초기화됩니다.
- 실운영 시 Redis 또는 Database 도입이 필요하며, 현재 Upstash Redis 연동 코드가 포함되어 있습니다.
